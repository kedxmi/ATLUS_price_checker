# ATLUS_price_checker
# SteamDB парсер

⚠  Этот проект предназначен в **образовательных целях**..

Этот парсер не имеет сложной структуры и функций,
он просто выдаёт актуальную цену на игры студии ATLUS

Это готовый набросок для страницы SteamDB для игр студии ATLUS, но помимо того
мой код можно спокойно интерпретировать под своб студию игр, за ценами которой вы следите.

## Использование

1. Зависимости:

```bash
pip install requests beautifulsoup4 lxml
```

Запуск:
```
python steam_parser.py
```

Важно !!!

Лучше всего для использования на локалке, взять свои данные из curl SteamDB.
